# 🐧 Linux Basics: Essential Commands for Support

### 1. File Management
- `ls -la`: List all files with permissions and owners.
- `cp / mv`: Copy or move files (essential for backups).
- `rm -i`: Safe removal of files (with confirmation).

### 2. System Monitoring
- `top` / `htop`: Check CPU and Memory usage in real-time.
- `df -h`: Check available disk space (Human-readable).
- `free -h`: Check RAM status.

### 3. Permissions (The Security Part)
- `chmod 700`: Owner can do everything, others - nothing.
- `chown`: Change file owner (critical for fixing access issues).

### 4. Networking
- `ip addr`: Check the machine's IP address.
- `ping`: Verify connectivity to a server or API.
- `netstat -tulpn`: See which ports are listening.
